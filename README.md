# 资金观测模型

这是一个面向 GitHub Pages 的静态资金观测面板，展示本仓库中的债券收益率和外汇报价 CSV 快照。

## 页面入口

- `index.html`：静态网页仪表盘
- `investing_global_bonds.csv`：全球主要国债收益率快照
- `investing_fx_quotes.csv`：外汇报价快照
- `investing_us_bonds.csv`：美国国债收益率快照
